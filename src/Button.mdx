# Button

## Contained Button

Contained Button was created based on [Material UI Contained Buttons](https://material-ui.com/components/buttons/#contained-buttons).

### Usage

```ts
import { ContainedButton } from '@iktakahiro/gradient-mui-button'
```

#### with Default Theme

<Playground>
  <ContainedButton color="primary">Primary</ContainedButton>
  <ContainedButton color="secondary">Secondary</ContainedButton>
  <ContainedButton>Default</ContainedButton>
  <ContainedButton color="primary" disabled={true}>Disabled</ContainedButton>
</Playground>

#### with Custom Theme

import { ThemeProvider } from '@material-ui/styles';

Contained Button inherits a defined [custom theme](https://material-ui.com/customization/color/).

```ts
// additional import statements to use a theme
import { createMuiTheme } from '@material-ui/core/styles';
import { teal, deepOrange } from '@material-ui/core/colors'
```

import { createMuiTheme } from '@material-ui/core/styles';
import { teal, deepOrange } from '@material-ui/core/colors';

<Playground>
  <ThemeProvider theme={createMuiTheme({
    palette: {
      primary: teal,
      secondary: {
        main: deepOrange[400]
      }
    }
  })}>
    <ContainedButton color="primary">Primary</ContainedButton>
    <ContainedButton color="secondary">Secondary</ContainedButton>
  </ThemeProvider>
</Playground>
